# SORI 작업 기록 — 2026년 6월 (커뮤니티 자동화 · 호스팅 정비)

> 이 문서는 SORI 플랫폼에 **무엇을, 왜, 어떻게 바꿨는지**를 개발을 모르는 사람도 바로 이해할 수 있게 정리한 기록입니다. 최신 작업(2026.06.22 ~ 06.24)을 중심으로 적었습니다.

---

## 0. SORI가 뭔가요 (1분 요약)

SORI는 **싱가포르에 사는 한국인을 위한 커뮤니티 앱**입니다. 뉴스, 자유게시판(커뮤니티), 한인 업소록, 부동산, 벼룩시장, 구인구직을 한 곳에 모았어요. 아직 정식 출시 전 **MVP(시제품) 단계**이고, 지금은 서버(백엔드) 없이 **브라우저에 데이터를 저장하는 방식**으로 동작합니다.

용어 2개만 알면 아래가 다 이해돼요:
- **GitHub = 창고** 📦 : 코드와 콘텐츠가 저장되는 곳 (`jhl9485/sori-platform`)
- **Netlify = 가게 진열** 🏪 : 창고의 내용을 실제 웹사이트로 띄워주는 곳 (자동 배포)

---

## 1. 커뮤니티 자동 글 — "사람이 활동하는 느낌" 만들기

**왜:** 커뮤니티 탭에 글이 적어 허전했어요. 정식 출시 때 방문자가 "다른 한인도 여기 쓰는구나" 하고 느끼게 하려고요.

**무엇:** 매일 아침(한국시간 9시쯤) **AI가 잔잔한 일상·정보 글 2개를 자동으로 써서**, 검토용 초안(PR)으로 올리고 진호님 메일로 알려줍니다. 진호님이 폰에서 보고 괜찮으면 **승인(병합)**하면 게시돼요.

**원칙:** 자극적이지 않게 / 정보성 글의 수치는 검증된 것만 / 특정 주제(COE·HDB BTO) 제외. (자세한 규칙은 `docs/community-auto-posting.md`)

**예:** 첫 자동 글 — [육아] "콘도 놀이터에서 또래 친구 사귀기, 저희 집 노하우 공유해요" 같은 톤.

---

## 2. 글 시간 표시 개선

**왜:** 글 시간이 `"2시간 전"`처럼 **글자로 고정**돼 있었어요. 자동 글이 쌓이면 전부 "2시간 전"으로 보여서 오히려 가짜 같았죠.

**무엇:** 글에 **실제 작성 날짜(createdAt)**를 넣고, 화면에서 **자동 계산**해 보여주게 바꿨어요. 이제 시간이 지나면 "방금 → 3시간 전 → 어제 → 3일 전 → 날짜"로 자연스럽게 바뀝니다.

---

## 3. 뉴스 검색 기능

뉴스 페이지에 **검색창**을 추가했어요. 제목·요약·카테고리·출처로 검색되고, 결과가 없으면 안내가 떠요. (커뮤니티·업소록과 같은 검색 디자인)

---

## 4. 커뮤니티 공지 정리

이미 지난 "P1 국제학생 신청 마감" 공지 배너를 제거했어요.

## 5. ⭐ 호스팅 문제와 결정 (가장 중요)

**무슨 일이 있었나:** 자동화(매일 뉴스 + 커뮤니티)로 사이트를 자주 새로 배포하다 보니, **Netlify 무료 한도(한 달에 약 20번 배포)를 3일 만에 다 써버렸어요.** 그래서 **새 업데이트가 라이브 사이트에 반영되는 게 멈췄습니다**(사이트 자체는 계속 켜져 있어요).

> 💡 핵심: **GitHub(창고)엔 새 뉴스·글이 다 들어와 있어요.** Netlify(진열)만 멈춘 거예요. 병합 버튼을 눌러도 사이트에 안 보이는 이유가 이것입니다.

**왜 그렇게 됐나:** 배포 1번에 비용(크레딧)이 들고, 무료 플랜은 월 한도가 있어요. 자동화로 매일 여러 번 배포하니 빠르게 소진됐어요.

**결정 (2026.06.24):** **7월 18일에 Netlify 무료 한도가 자동으로 리셋**되니, 그때까지 기다리기로 했어요(추가 비용 0). 그동안 자동 글은 GitHub에 계속 쌓이고, 7/18 이후 한 번에 반영됩니다.

**Vercel로 옮기는 건?** 더 후한 무료 서비스인 Vercel로 이전하려 했지만 — 신규 가입이 **유료 체험(카드 등록)을 강제**하고, Vercel 무료(Hobby)는 **"비상업용·개인 프로젝트만"** 허용하는 약관이라(SORI는 향후 수익 모델이 있음) **보류**했어요. (`vercel.json` 설정은 나중 재시도 대비해 남겨둠)

**안전장치:** 7/18 아침에 "이제 배포 재개하세요"라는 **알림 메일이 자동 발송**되도록 클라우드에 예약해뒀어요. (컴퓨터가 꺼져 있어도 작동)

## 6. 커뮤니티 자동화 버그 수정
